Para criar o Virtual Environment:
- py -3.11 -m venv .venv (dentro do diretório certo, .venv é nome)
- python3 manage.py runserver
 
Para ativar o VEnv: 
- Windows: .venv\Scripts\activate
- Linux: source .venv/bin/
- source .eventif/bin/activate

Para desativar:
- deactivate

Para ativar servidor: 
- py -3.11 manage.py runserver

Para executar teste: 
- py -3.11 manage.py test

Para criar app:
- py -3.11 manage.py startapp nomeApp

Para instalar os requirements.txt
- py -m pip install -r /path/to/requirements.txt



Fazer commit:
git status
